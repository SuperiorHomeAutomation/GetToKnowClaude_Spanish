## Amazon-Safe Writing Rules

Amazon KDP uses automated keyword scanning that flags AI-related phrases regardless of context.
The following rules apply to ALL Amazon-safe versions only.

### Words and phrases to AVOID in Amazon versions:

| Avoid | Use instead |
|-------|------------|
| Claude wrote | the author wrote |
| Claude generated | produced using AI tools |
| AI-generated content | AI-assisted editing |
| written with Claude | developed with AI assistance |
| Claude as a thinking partner | Claude as a productivity tool |
| writing collaborator | editing tool |
| Claude improved my English | AI tools helped with editing |
| Claude drafted sections | sections were refined with AI tools |
| ghost-written | (avoid entirely) |
| collaborated on | developed |
| The Role of AI in This Book | About This Book |
| Foreword by Claude | (remove foreword entirely from Amazon version) |
| [book] exists because Claude made it possible | the right tools made it possible |

### CRITICAL — Foreword must be REMOVED ENTIRELY from Amazon versions:
The foreword is written by Claude and signed by Claude. Amazon's scanner flags this.
Do NOT include it in any Amazon-safe version. Remove the entire foreword section.
The Preface and all other sections remain unchanged.

### Safe phrases that are fine for Amazon:
- Using Claude to edit books, Claude as a tool, How to use Claude ✅
- Claude AI assistant, AI productivity tool, AI assistance for editing ✅
- Claude helped improve my English, Claude as a writing tool ✅
- writing partner (when teaching readers how to use Claude) ✅

### Key distinction:
- "AI wrote this book" — FLAGGED
- "Author used AI tools to edit and improve the book" — SAFE
